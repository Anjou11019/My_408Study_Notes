# 第三章

[[第三章 内存管理]]

**重点**
 - 三种分配方式是否有内、外部碎片

==**重要考点**==：计算机中内存块的数量 -> 页表项中块号占多少字节

**重要考点/高频考点**：$页面大小\leftrightarrow 页内偏移量位数 \leftrightarrow 逻辑地址结构$ 
 - 如果题目中页面大小不是2的整数次幂，这种情况就需要古法计算
    - 页号  = 逻辑地址/页面长度（取整）
    - 页内偏移量 = 逻辑地址%页面长度（取余）

**重要考点/高频考点**：地址变换过程
 1. 根据逻辑地址算出页号、页内偏移量
 2. 页号的合法性检查（与页表长度对比）
 3. 若页号合法，再根据页表起始地址、页号找到对应页表项
 4. 根据页表项中记录的内存块号、页内偏移量得到最终的物理地址
 5. 访问物理内存对应的内存单元

**重要考点**：
